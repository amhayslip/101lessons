HTML Vocabulary

HTML Element:

An HTML element is an individual component of an HTML document or web page, once this has been parsed into the Document Object Model. For example a paragraph "tag", creates a paragraph element once it is on a web-page.

HTML Tag:

An HTML tag is actual syntax of HTML which make HTML Elements. The example below is a "paragraph" or "p" "tag". The first is considered an opening tag and the second is considered a closing tag.

<p></p>

CSS Vocabulary

CSS Selector:

A CSS Selector is the way in which HTML elements are selected to style. The following example is a CSS selector that is selecting an element with the class, "container".

.container

CSS Property:

A CSS property is an identifier, that is a human-readable name, that defines which feature is considered. In the example below, "font-family" is the property, and "sans-serif" is the value.

p {
  font-family: "sans-serif";
}

Javascript Vocabulary

Function:

A function is a Javascript procedure - a set of statements that performs a task or calculates a value. The following is an example:

var greet = function () {
  console.log("hello");
}

Function Argument:

An argument is a variable that can be passed to functions and made available inside their scope. The following is a function with one argument called greeting:

var greet = function (greeting) {
  console.log(greeting);
}

Javascript Object:

Javascript objects are data-structures that have key-value pairs. The following is an example of an object:

var car = {
  color: "red",
}

Javascript Object Property:

Object properites are the "key" in the kay-value pair. They are the descriptors of objects. In the example below, "name" is a property, and "Bob" is it's value:

var person = {
  name: "Bob"
}

Javascript Method:

A Javascipt "Method" is a function that happens to also be the propety of an object. In the example below, "go" is a method of our car object:

var car = {
  go: function () {
    console.log("going!");
  }
}

Constructor Function:

A Javascript Constructor Function is a special kind of function that creates (or "constructs") objects. They can be identified by their capitalized variable name, and reference to "this". For example:

var Car = function () {
  this.name = "ferrari";
}
