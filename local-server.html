Intro

This will guide your through setting up a local server as mentioned in the Searchstagram project. Have you project opened and completed to the end of Searchstagaram Part 3.

Using Gulp

If you have a mac/linux machine, there are other simpler options than using gulp and you skip this and read the one called Using Java below if you want. There are other options too with a windows machine too, but they're probably not simpler than this. Now let's look at those files we skipped over last time:

node_modules

These are basically dependency files that we've installed using NPM. NPM stands for Node Package Manager and it's a way that we can use Node to install and keep track of "packages" (code other people have written that could be useful to us... like Bootstrap or jQuery). In order to use NPM you have to download NODE. Go here to do that.

package.json

This is a JSON file (Javascript Object Notation) that containers all the things we have installed inside of our node_modules folder.

gulpfile.js

Gulp is one of those "packages" that we've installed into the "node_modules" folder using NPM. It's a task manager that will... perform tasks when we tell it to. One of those tasks is going to be serving our index.html file at http://localhost:4000.

Using Gulp with a Mac/Linux Machine

We have one "gulp" task already built and it will run our server. All we have to do is tell it to. We'll do this through the "command line". Open up the program on your computer called "terminal". You will see the following:

Now type in...

ls

...and press Enter. This will list all the files in your current directory. We want to navigate to where our Searchstragram folder is. To move forward into a folder, we use cd + foldername. For example, if I wanted to move into the Desktop folder I would type:

cd desktop

And if I wanted to back up one, I would type:

cd ..

Using these commands, navigate to your searchstagram folder and type:

npm install

Then type:

sudo npm install gulp

This will prompt you for your computer's password. Type it in and press enter. And finally type:

gulp

And there it is! Your server should be running at http://localhost:4000
