Intro

So far we've been writing a bunch of code that cannot really be used (think, the Library) because there's no user interface. Well, let's change that and let's start with getting input from the user. Check out this example, and we'll break it down.

The <input> Tag and .val() Method

The input tag created for us our blank text box.

<input></input>

We can get the "value" of the text inside of the text box by using jQuery's .val() method. The following would return a string of text from an input tag: 

$('input').val()

The .append() Method

The .append() method adds or "appends" HTML to a given element. For example:

$('div').append('hello world');

The code above would append 'hello world' to the generic div element.
